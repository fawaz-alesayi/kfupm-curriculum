KFUPM Curriculum Admin page is a functional page to let the admin modify, add, remove or update courses information in the System. Admin can choose any major and change the flowchart or courses by clicking on function button or text to process changing easily ! Plus, Admin can go to courses page and modify all information right there by clicking on the button, or text of the paragraph that needs to be changed. Admin pages are the same of default pages, only background color is different & there're functions so the Admin can use them to do modification easily, and successfully.
